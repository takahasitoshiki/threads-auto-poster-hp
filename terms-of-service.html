利用規約

1. サービスの概要

Threads Auto Poster（以下「本サービス」）は、Google カレンダーの予定を Meta Threads へ自動投稿する Google Workspace Add-on です。ユーザーはカレンダーに予定を作成するだけで、指定した日時に Threads への投稿を自動的に行うことができます。

2. 利用条件

本サービスを利用するには、以下の条件を満たす必要があります:

  • Google アカウントを保有していること
  • Meta Threads アカウントを保有していること
  • Meta for Developers で Threads API のアクセストークンを取得していること
  • 本利用規約およびプライバシーポリシーに同意すること

  • API の仕様変更に伴うサービスの中断
  • Google Apps Script の時間トリガーの精度（±15分の誤差が生じる場合があります）
