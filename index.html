よくある質問

投稿時刻は正確ですか？

Google Apps Script の仕様により、最大 ±15分の誤差が生じます。

スマホから予約できますか？

はい。「Threads投稿」カレンダーに予定を作成するだけなので、スマホの Google カレンダーアプリからも予約できます。

1日に何件投稿できますか？

Threads API の制限で、24時間あたり最大 250件です。

トークンの有効期限は？

60日間です。本ツールは45日経過時に自動リフレッシュを行います。
