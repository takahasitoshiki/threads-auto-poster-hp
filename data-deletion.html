データ削除について

保存されるデータ

Threads Auto Poster は、以下のデータを Google Apps Script UserProperties に保存します:

データ項目説明
Threads ユーザーIDAPI 呼び出しに使用する数値ID
アクセストークンThreads API 認証用トークン
トークン保存日時自動リフレッシュの判定に使用
月次投稿カウントダッシュボード表示用の統計

重要: 本アプリは独自のサーバーやデータベースを持ちません。すべてのデータは Google のインフラ上（UserProperties）にのみ保存されます。

データの削除方法

方法 1: Add-on 設定画面から削除（推奨）

  1. Google カレンダーを開く
  2. 右サイドバーの Threads Auto Poster アイコンをクリック
  3. 「設定」を開く
  4. 「接続を解除」ボタンをクリック

→ ユーザーID、アクセストークン、保存日時、月次統計データ、およびすべてのトリガーが即座に削除されます。

方法 2: Add-on をアンインストール

  1. Google カレンダーを開く
  2. 右サイドバーの Threads Auto Poster アイコンを右クリック
  3. 「アンインストール」を選択

→ Add-on に関連するすべての UserProperties とトリガーが Google によって削除されます。

方法 3: メールでの削除リクエスト

上記の方法で削除できない場合は、以下のメールアドレスにご連絡ください:

[YOUR_EMAIL]

件名に「データ削除リクエスト」と記載し、ご利用の Google アカウントのメールアドレスをお知らせください。

削除の確認

「接続を解除」を実行した後、再度 Add-on を開くと初期設定画面が表示されます。これにより、すべてのデータが削除されたことを確認できます。

Data Deletion

Data We Store

Threads Auto Poster stores the following data in Google Apps Script UserProperties:

DataDescription
Threads User IDNumeric ID used for API calls
Access TokenToken for Threads API authentication
Token Save TimestampUsed for automatic refresh scheduling
Monthly Post CountStatistics for dashboard display

Important: This app does not operate its own servers or databases. All data is stored exclusively on Google's infrastructure (UserProperties).

How to Delete Your Data

Method 1: Disconnect from Add-on Settings (Recommended)

  1. Open Google Calendar
  2. Click the Threads Auto Poster icon in the right sidebar
  3. Open "Settings"
  4. Click the "Disconnect" button

→ Your User ID, access token, save timestamp, monthly statistics, and all triggers are immediately deleted.

Method 2: Uninstall the Add-on

  1. Open Google Calendar
  2. Right-click the Threads Auto Poster icon in the right sidebar
  3. Select "Uninstall"

→ All UserProperties and triggers associated with the Add-on are deleted by Google.

Method 3: Email Deletion Request

If you are unable to delete your data using the methods above, please contact us at:

[YOUR_EMAIL]

Include "Data Deletion Request" in the subject line and provide the email address of your Google account.

Verifying Deletion

After disconnecting, reopen the Add-on. If the initial setup screen appears, all your data has been successfully deleted.
